Le siège auto NANIA Befix s'adapte à tous les véhicules dotés d'une ceinture de sécurité à 3 points. ## Installation du siège auto Nania Befix

### Assemblage du siège

Le siège Nania est constitué de deux éléments distincts : l'assise, également connue sous le nom de rehausseur, et le dossier. Le montage est très simple et se compose de trois étapes simples :

étape 1  : Insérez les guides du dossier dans la partie arrière du siège.
étape 2  : Relevez le dossier à la verticale.
étape 3  : Exercez une pression sur le dossier pour vous assurer que les deux supports s'emboîtent correctement.

### Installation sur le siège du véhicule

Bien que le nom du **siège auto Nania Befix** puisse prêter à confusion, il ne nécessite pas de système d'installation ISOFIX. Il doit uniquement être installé en utilisant les ceintures de sécurité à trois points du véhicule.
Nous vous rappelons l'importance de toujours attacher le siège avec les sangles de voiture, même en l'absence de l'enfant dans le véhicule, afin de garantir la sécurité.
